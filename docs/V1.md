# Lego Stylized Nature V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a WebGPU + TSL foundation for a Lego-style stylized nature scene with noise-based terraced terrain, hand-guided biome regions, a single shared Blender brick asset rendered through instancing with per-instance colors, Blender-exported nature prefabs, and four polished biomes viewed from an orthographic bird's-eye camera.

**Architecture:** V1 uses a fixed-size mainland instead of infinite terrain streaming. Terrain height is generated from seeded simplex noise, biome ownership is generated from hand-guided cell regions. Heights blend smoothly across biome borders; brick colors use hard-cut dithering. All terrain bricks share one 2x2 brick geometry and render as a single opaque `InstancedMesh` plus one translucent water `InstancedMesh`. Hidden bricks are culled at generation time with a closed-form heightfield rule. Prefabs are Blender GLBs (one GLB per variant, native materials) rendered as per-sub-mesh `InstancedMesh` groups sharing transform lists. `World` is an orchestrator; terrain, biome, brick rendering, prefab placement, materials, and debug panels are separated into focused modules.

**Tech Stack:** Vite, Three.js WebGPU (`three/webgpu`), TSL (`three/tsl`), `simplex-noise`, Tweakpane, GLTF/GLB assets exported from Blender.

---

## V1 Scope

V1 builds a controllable showcase island/mainland, not a full open-world engine.

Included:

- Fixed terrain size, recommended default `96 x 96` terrain cells.
- Seeded simplex-noise FBM height generation with sea-clip flattening and floor quantization (terraced).
- Four biomes: `grassland`, `forest`, `desert`, `tundra`.
- Hand-guided biome regions; smooth height blending at borders, hard-cut + dithered brick colors at borders.
- One shared `brick-2x2.glb` geometry for all terrain and water bricks; colors assigned per instance.
- Visible-layer culling: bricks fully occluded on all six sides are never generated.
- No underwater terrain bricks; water is a single top layer of translucent water bricks.
- Blender-exported nature prefabs (trees, flowers, cactus, rocks), one GLB per variant, native Blender materials, 1x1 cell footprint.
- `InstancedMesh` rendering for terrain bricks, water bricks, and every prefab sub-mesh. No `scene.clone()` in the render path.
- TSL materials: Lego brick material with per-instance color, water material with translucency and time-driven color ripple.
- Orthographic bird's-eye camera with constrained orbit.
- Debug panels for seed, terrain height, biome layout, placement density, and material parameters.

V1 excludes:

- Infinite chunk streaming.
- Player controller, combat, inventory, mining, or survival mechanics.
- Climate simulation based on temperature and humidity.
- Procedural mesh generation for cactus, bamboo, trees, flowers, or Lego bricks.
- Water vertex animation (color ripple only).
- Full biome ecology rules.
- Runtime occlusion culling or LOD (generation-time culling only).

- The Geometry Nodes terrain proof is documented in `docs/2026-06-08-geometry-nodes-procedural-terrain.md`.

Important design constraints:

- Blender provides exactly one terrain brick asset (`brick-2x2.glb`) plus nature prefab GLBs. Runtime code selects, places, transforms, colors, and shades assets; it does not procedurally model Lego pieces.
- Asset export conventions (must hold for every GLB):
  - Origin at **bottom center** of the model.
  - One terrain cell = one 2x2 brick footprint. The brick's X/Z size defines `cellSize`; runtime normalizes scale from the brick's bounding box.
  - Prefabs occupy a 1x1 cell footprint and sit flush on a brick's top face at `y = 0` local.
  - +Y up after glTF export.
- The camera is orthographic and bird's-eye. Users never see the terrain from below; the underside of the heightfield needs no geometry.

## Data Flow

```txt
WorldConfig
  -> Resources loads brick-2x2.glb and prefab variant GLBs
  -> BiomeRegistry loads four biome definitions
  -> BiomeMaskGenerator creates biome id and biome weights per cell
  -> TerrainGenerator creates HeightField from seeded simplex FBM
       (heights blended smoothly from biome weights)
  -> SurfaceClassifier labels cells as water, shore, slope, land
  -> LayeredTerrainBuilder emits visible-brick placements only
       (closed-form occlusion culling, water columns skipped)
  -> BrickColorResolver assigns a dithered biome layer color per brick
  -> TerrainBrickRenderer fills one opaque InstancedMesh (matrix + color per instance)
  -> WaterBrickRenderer fills one translucent InstancedMesh at the water level
  -> PrefabPlacer scatters prefab transforms on valid top surfaces
       and builds per-sub-mesh InstancedMesh groups per variant
  -> Debug panels expose seed, rules, and material controls
```

## Naming Conventions

- Biome ids use lower camel case: `grassland`, `forest`, `desert`, `tundra`.
- Brick color roles per biome: `surface`, `subsurface`, `deep`, `shore`.
- Prefab ids describe biome and object family: `desertCactusSmall`, `forestPine`, `grasslandFlowerWhite`, `tundraSnowRock`. Variants live inside a prefab entry, each variant maps to one GLB source.
- Brick placement records use world cell coordinates: `{ x, y, z, layer, biomeId }` where `y` is a layer index.
- Runtime classes use PascalCase filenames and named methods with explicit responsibilities.

```js
import * as THREE from 'three/webgpu'
import { OrbitControls } from 'three/addons/controls/OrbitControls.js'

export default class WorldCamera {
    /**
     * @param {HTMLCanvasElement} canvas
     * @param {import('../systems/Sizes.js').default} sizes
     */
    constructor(canvas, sizes) {
        this.sizes = sizes
        this.canvas = canvas

        this._frustumHeight = 14

        const aspect = sizes.width / sizes.height
        this.instance = new THREE.OrthographicCamera(
            -this._frustumHeight * aspect * 0.5,
            this._frustumHeight * aspect * 0.5,
            this._frustumHeight * 0.5,
            -this._frustumHeight * 0.5,
            0.1,
            200
        )
        this.instance.position.set(40, 40, 40)

        this.controls = new OrbitControls(this.instance, canvas)
        this.controls.enableDamping = true
        this.controls.maxPolarAngle = Math.PI / 3
        this.controls.minZoom = 0.5
        this.controls.maxZoom = 6
    }

    lookAt(target) {
        this.controls.target.copy(target)
        this.controls.update()
    }

    resize() {
        const aspect = this.sizes.width / this.sizes.height
        this.instance.left = -this._frustumHeight * aspect * 0.5
        this.instance.right = this._frustumHeight * aspect * 0.5
        this.instance.top = this._frustumHeight * 0.5
        this.instance.bottom = -this._frustumHeight * 0.5
        this.instance.updateProjectionMatrix()
    }

    update() {
        this.controls.update()
    }

    /**
     * @param {import('../utils/debug.js').default} debug
     */
    debuggerInit(debug) {
        const folder = debug.addFolder({ title: 'Camera', expanded: false })
        if (!folder) {
            return
        }
        folder.addBinding(this, '_frustumHeight', { min: 4, max: 60, step: 1, label: 'Frustum H' }).on('change', () => {
            this.resize()
        })
    }

    dispose() {
        this.controls.dispose()
    }
}
```

- [ ] **Step 3: Create deterministic random helpers**

Create `src/utils/random.js`. `mulberry32` seeds the simplex noise; `hash2`/`random01` drive per-cell decisions (dithering, variant picks, rotations) without any call-order dependency.

Target implementation:

```js
export function mulberry32(seed) {
    let a = seed >>> 0
    return function () {
        a |= 0
        a = (a + 0x6D2B79F5) | 0
        let t = Math.imul(a ^ (a >>> 15), 1 | a)
        t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) ^ t
        return ((t ^ (t >>> 14)) >>> 0) / 4294967296
    }
}

export function hash2(x, z, seed = 0) {
    let h = seed ^ Math.imul(x + 374761393, 668265263) ^ Math.imul(z + 69069, 2246822519)
    h = (h ^ (h >>> 13)) >>> 0
    h = Math.imul(h, 1274126177) >>> 0
    return (h ^ (h >>> 16)) >>> 0
}

export function random01(x, z, seed = 0) {
    return hash2(x, z, seed) / 4294967295
}

export function pickWeighted(entries, randomValue) {
    let total = 0
    for (const entry of entries) {
        total += entry.weight
    }

    if (total <= 0) {
        return entries[0]?.value
    }

    let cursor = randomValue * total
    for (const entry of entries) {
        cursor -= entry.weight
        if (cursor <= 0) {
            return entry.value
        }
    }

    return entries[entries.length - 1]?.value
}

export function snapValue(value, step) {
    return Math.round(value / step) * step
}
```

```js
export default class BiomeMaskGenerator {
    constructor(config) {
        this.config = config
    }

    generate() {
        const { width, depth } = this.config.terrain
        const cells = []

        for (let z = 0; z < depth; z++) {
            const row = []
            for (let x = 0; x < width; x++) {
                row.push(this.getCellBiome(x, z))
            }
            cells.push(row)
        }

        return cells
    }

    getCellBiome(x, z) {
        const regions = this.config.biomes.regions
        const scores = regions.map((region) => {
            const dx = x - region.center[0]
            const dz = z - region.center[1]
            const distance = Math.sqrt(dx * dx + dz * dz)
            const normalized = distance / region.radius
            const score = Math.max(0, 1 - normalized) * region.weight
            return { id: region.id, score }
        }).filter((entry) => entry.score > 0)

        if (scores.length === 0) {
            return { biomeId: 'grassland', weights: { grassland: 1 } }
        }

        scores.sort((a, b) => b.score - a.score)
        const selected = scores.slice(0, 2)
        const total = selected.reduce((sum, entry) => sum + entry.score, 0)
        const weights = {}

        for (const entry of selected) {
            weights[entry.id] = entry.score / total
        }

        return { biomeId: selected[0].id, weights }
    }
}
```

```js
    constructor(registry) {
        this.registry = registry
    }

    blendTerrainParam(weights, paramName, fallback = 0) {
        let value = 0
        for (const [biomeId, weight] of Object.entries(weights)) {
            const biome = this.registry.get(biomeId)
            value += (biome.terrain[paramName] ?? fallback) * weight
        }
        return value

        const entries = Object.entries(weights).map(([biomeId, weight]) => ({ value: biomeId, weight }))
        return pickWeighted(entries, random01(x, z, seed + 7919))
    }

    getDominantBiome(cell) {
        return this.registry.get(cell.biomeId)
    }
```

```js
export default class HeightField {
    constructor(width, depth) {
        this.width = width
        this.depth = depth
        this.values = Array.from({ length: depth }, () => Array.from({ length: width }, () => 0))
    }

    get(x, z) {
        return this.values[z]?.[x] ?? 0
    }

    set(x, z, value) {
        if (this.values[z]) {
            this.values[z][x] = value
        }
    }

    getSlope(x, z) {
        const center = this.get(x, z)
        const neighbors = [
            this.get(x + 1, z),
            this.get(x - 1, z),
            this.get(x, z + 1),
            this.get(x, z - 1)
        ]
        return Math.max(...neighbors.map((height) => Math.abs(center - height)))
    }
}
```

Target implementation:

```js
import { createNoise2D } from 'simplex-noise'
import HeightField from './HeightField.js'
import TerrainMap from './TerrainMap.js'
import SurfaceClassifier from './SurfaceClassifier.js'
import { mulberry32 } from '../../utils/random.js'

export default class TerrainGenerator {
    constructor({ config, biomeMaskGenerator, biomeBlender }) {
        this.config = config
        this.biomeMaskGenerator = biomeMaskGenerator
        this.biomeBlender = biomeBlender
        this.surfaceClassifier = new SurfaceClassifier(config)
    }

    generate() {
        this.noise2D = createNoise2D(mulberry32(this.config.seed))
        const biomeCells = this.biomeMaskGenerator.generate()
        const heightField = this.generateHeightField(biomeCells)
        const surfaceCells = this.surfaceClassifier.classify(heightField)
        return new TerrainMap({ heightField, biomeCells, surfaceCells })
    }

    generateHeightField(biomeCells) {
        const terrain = this.config.terrain
        const field = new HeightField(terrain.width, terrain.depth)

        for (let z = 0; z < terrain.depth; z++) {
            for (let x = 0; x < terrain.width; x++) {
                const biomeCell = biomeCells[z][x]
                const heightOffset = this.biomeBlender.blendTerrainParam(biomeCell.weights, 'heightOffset', 0)
                const heightMagnitude = this.biomeBlender.blendTerrainParam(biomeCell.weights, 'heightMagnitude', 1)

                const n01 = 0.5 + 0.5 * this.fbm(x, z)
                const shaped = Math.max(0, Math.min(1, (n01 - terrain.seaClip) / (1 - terrain.seaClip)))
                const height = Math.floor(shaped * terrain.maxHeight * heightMagnitude + terrain.waterLevel + heightOffset)

                field.set(x, z, Math.max(0, Math.min(terrain.maxHeight, height)))
            }
        }

        return field
    }

    fbm(x, z) {
        const terrain = this.config.terrain
        let value = 0
        let amplitude = 1
        let frequency = 1 / terrain.noiseScale
        let totalAmplitude = 0

        for (let octave = 0; octave < terrain.noiseOctaves; octave++) {
            value += this.noise2D(x * frequency, z * frequency) * amplitude
            totalAmplitude += amplitude
            amplitude *= terrain.noiseGain
            frequency *= terrain.noiseLacunarity
        }

        return value / totalAmplitude
    }
}
```

**Files:**

- Create: `src/world/bricks/BrickGeometry.js`

Rendering contract for this task:

- All opaque terrain bricks live in **one** `InstancedMesh` (one draw call) sharing the `brick2x2Model` geometry and the Lego TSL material; per-brick color uses `setColorAt`.
- Placements contain **visible bricks only**. Because terrain is a heightfield (columns filled bottom-to-top, no overhangs), full 6-neighbor voxel occlusion reduces to a closed-form rule per column:

```txt
m = min(effectiveHeight of 4 lateral neighbors)
visible bricks of column (x, z) with height h = y in [min(h, m + 1), h]

effectiveHeight(x, z):
  out of grid          -> -1          (island edge walls render to y = 0)
  water column (h<=WL) -> waterLevel  (no bricks below the water line are visible
                                       from the bird's-eye view through 0.82-opacity water)
  land column          -> h
```

- Water columns produce **no terrain bricks at all** (decided: no underwater terrain; orthographic bird's-eye only).
- Estimated instance count: ~9216 columns x (1 top brick + small lateral exposure) ≈ 10k-20k, versus ~92k for naive full columns.

```js
import * as THREE from 'three/webgpu'

export function extractBrickGeometry(gltf, cellSize) {
    let sourceMesh = null
    gltf.scene.traverse((child) => {
        if (child.isMesh && !sourceMesh) {
            sourceMesh = child

    if (!sourceMesh) {
        return null

    const geometry = sourceMesh.geometry.clone()
    geometry.computeBoundingBox()
    const size = new THREE.Vector3()
    geometry.boundingBox.getSize(size)

    const scale = cellSize / size.x
    geometry.scale(scale, scale, scale)
```

```js
export default class LayeredTerrainBuilder {
    constructor({ config }) {
        this.config = config
    }

    buildPlacements(terrainMap) {
        const placements = []
        const { width, depth, waterLevel } = this.config.terrain

        const effectiveHeight = (x, z) => {
            if (x < 0 || z < 0 || x >= width || z >= depth) {
                return -1
            }
            const h = terrainMap.getHeight(x, z)
            return h <= waterLevel ? waterLevel : h
        }

        for (let z = 0; z < depth; z++) {
            for (let x = 0; x < width; x++) {
                const surfaceCell = terrainMap.getSurfaceCell(x, z)
                if (surfaceCell.isWater) {
                    continue
                }

                const biomeCell = terrainMap.getBiomeCell(x, z)
                const h = surfaceCell.height
                const m = Math.min(
                    effectiveHeight(x + 1, z),
                    effectiveHeight(x - 1, z),
                    effectiveHeight(x, z + 1),
                    effectiveHeight(x, z - 1)
                )
                const yStart = Math.min(h, m + 1)

                for (let y = yStart; y <= h; y++) {
                    const layer = y === h
                        ? 'surface'
                        : y >= h - 2
                            ? 'subsurface'
                            : 'deep'
                    placements.push({ x, y, z, layer, biomeCell, surfaceCell })
                }
            }
        }

        return placements
    }
}
```

Create `src/world/bricks/TerrainBrickRenderer.js`. Capacity is preallocated and reused across regenerations; only when placements outgrow capacity is the `InstancedMesh` rebuilt. The brick geometry is shared with the GLB and must not be disposed here.

Target implementation:

```js
import * as THREE from 'three/webgpu'
import { createLegoMaterial } from '../../materials/tsl/legoMaterial.js'

export default class TerrainBrickRenderer {
    constructor({ config, brickGeometry }) {
        this.config = config
        this.brickGeometry = brickGeometry
        this.material = createLegoMaterial()
        this.group = new THREE.Group()
        this.group.name = 'TerrainBricks'
        this.mesh = null
        this.capacity = 0
    }

    build(placements, colorResolver) {
        const { cellSize, layerHeight } = this.config.terrain

        if (!this.mesh || placements.length > this.capacity) {
            this.mesh?.dispose()
            this.group.remove(this.mesh)
            this.capacity = Math.ceil(placements.length * 1.2)
            this.mesh = new THREE.InstancedMesh(this.brickGeometry, this.material, this.capacity)
            this.mesh.name = 'TerrainBrickInstances'
            this.group.add(this.mesh)
        }

        const matrix = new THREE.Matrix4()
        const color = new THREE.Color()

        placements.forEach((p, i) => {
            matrix.setPosition(
                (p.x + 0.5) * cellSize,
                p.y * layerHeight,
                (p.z + 0.5) * cellSize
            )
            this.mesh.setMatrixAt(i, matrix)

            color.set(colorResolver.resolve({
                biomeCell: p.biomeCell,
                surfaceCell: p.surfaceCell,
                layer: p.layer,
                x: p.x,
                z: p.z
            }))
            this.mesh.setColorAt(i, color)
        })

        this.mesh.count = placements.length
        this.mesh.instanceMatrix.needsUpdate = true
        if (this.mesh.instanceColor) {
            this.mesh.instanceColor.needsUpdate = true
        }

        return this.group
    }

    dispose() {
        this.mesh?.dispose()
        this.material.dispose()
        this.group.parent?.remove(this.group)
        this.group.clear()
        this.mesh = null
        this.capacity = 0
    }
}
```

```js
import { random01, snapValue, pickWeighted } from '../../utils/random.js'

export function canPlacePrefab(rule, surfaceCell) {
    if (surfaceCell.isWater || surfaceCell.isShore) {
        return false
    }
    if (rule.minHeight !== undefined && surfaceCell.height < rule.minHeight) {
        return false
    }
    if (rule.maxSlope !== undefined && surfaceCell.slope > rule.maxSlope) {
        return false
    }
    return true
}

export function pickVariantIndex(manifestEntry, x, z, seed) {
    const entries = manifestEntry.variants.map((variant, index) => ({ value: index, weight: variant.weight }))
    return pickWeighted(entries, random01(x, z, seed + 131))
}

export function makePrefabTransform({ x, z, height, manifestEntry, config, seed }) {
    const { cellSize, layerHeight } = config.terrain
    const { rotationStep, scaleStep } = config.placement

    const rotationY = manifestEntry.randomRotation
        ? snapValue(random01(x, z, seed + 17) * Math.PI * 2, rotationStep)
        : 0

    let scale = 1
    if (manifestEntry.randomScale) {
        const [min, max] = manifestEntry.randomScale
        scale = snapValue(min + random01(x, z, seed + 23) * (max - min), scaleStep)
    }

    return {
        position: [(x + 0.5) * cellSize, (height + 1) * layerHeight, (z + 0.5) * cellSize],
        rotationY,
        scale
    }
}
```

- [ ] **Step 3: Create PrefabPlacer with shared-matrix instancing**

Create `src/world/prefabs/PrefabPlacer.js`.

Target implementation:

```js
import * as THREE from 'three/webgpu'
import { random01 } from '../../utils/random.js'
import { canPlacePrefab, pickVariantIndex, makePrefabTransform } from './placementRules.js'

export default class PrefabPlacer {
    constructor({ config, biomeRegistry, prefabRegistry }) {
        this.config = config
        this.biomeRegistry = biomeRegistry
        this.prefabRegistry = prefabRegistry
        this.group = new THREE.Group()
        this.group.name = 'BiomePrefabs'
    }

    build(terrainMap) {
        this.dispose()
        this.group = new THREE.Group()
        this.group.name = 'BiomePrefabs'

        const buckets = this.collectTransforms(terrainMap)

        for (const [key, transforms] of buckets) {
            const [prefabId, variantIndex] = key.split(':')
            const gltf = this.prefabRegistry.getVariantAsset(prefabId, Number(variantIndex))
            if (!gltf?.scene) {
                continue
            }
            this.group.add(this.buildVariantInstances(gltf.scene, transforms))
        }

        return this.group
    }

    collectTransforms(terrainMap) {
        const buckets = new Map()
        const { width, depth } = this.config.terrain
        const seed = this.config.seed

        for (let z = 0; z < depth; z++) {
            for (let x = 0; x < width; x++) {
                const biomeCell = terrainMap.getBiomeCell(x, z)
                const biome = this.biomeRegistry.get(biomeCell.biomeId)
                const surfaceCell = terrainMap.getSurfaceCell(x, z)

                for (const rule of biome.prefabs) {
                    if (!canPlacePrefab(rule, surfaceCell)) {
                        continue
                    }
                    if (random01(x, z, seed + rule.id.length) > rule.density) {
                        continue
                    }

                    const prefab = this.prefabRegistry.get(rule.id)
                    if (!prefab) {
                        continue
                    }

                    const variantIndex = pickVariantIndex(prefab.entry, x, z, seed)
                    const transform = makePrefabTransform({
                        x,
                        z,
                        height: surfaceCell.height,
                        manifestEntry: prefab.entry,
                        config: this.config,
                        seed
                    })

                    const key = `${rule.id}:${variantIndex}`
                    if (!buckets.has(key)) {
                        buckets.set(key, [])
                    }
                    buckets.get(key).push(transform)
                    break
                }
            }
        }

        return buckets
    }

    buildVariantInstances(sourceScene, transforms) {
        sourceScene.updateMatrixWorld(true)

        const variantGroup = new THREE.Group()
        const instanceMatrix = new THREE.Matrix4()
        const composed = new THREE.Matrix4()
        const position = new THREE.Vector3()
        const quaternion = new THREE.Quaternion()
        const scale = new THREE.Vector3()
        const yAxis = new THREE.Vector3(0, 1, 0)

        sourceScene.traverse((child) => {
            if (!child.isMesh) {
                return
            }

            const mesh = new THREE.InstancedMesh(child.geometry, child.material, transforms.length)
            transforms.forEach((t, i) => {
                position.fromArray(t.position)
                quaternion.setFromAxisAngle(yAxis, t.rotationY)
                scale.setScalar(t.scale)
                instanceMatrix.compose(position, quaternion, scale)
                composed.multiplyMatrices(instanceMatrix, child.matrixWorld)
                mesh.setMatrixAt(i, composed)
            })
            mesh.instanceMatrix.needsUpdate = true
            variantGroup.add(mesh)
        })

        return variantGroup
    }

    dispose() {
        this.group?.traverse((child) => {
            if (child.isInstancedMesh) {
                child.dispose()
            }
        })
```

```js
import { color, time, positionWorld, sin, uniform } from 'three/tsl'

export function createWaterMaterial(waterConfig) {
    const material = new THREE.MeshStandardNodeMaterial()

    const uRippleStrength = uniform(waterConfig.rippleStrength)
    const uRippleSpeed = uniform(waterConfig.rippleSpeed)

    const ripple = sin(
        time.mul(uRippleSpeed)
            .add(positionWorld.x.mul(2.1))
            .add(positionWorld.z.mul(1.7))

    material.opacity = waterConfig.opacity
    material.depthWrite = false
    material.roughness = 0.25

    material.userData.uniforms = { uRippleStrength, uRippleSpeed }

    return material
}
```

Water rendering contract (decided in review):

- Only the top layer exists. No bricks below the water line anywhere in the scene (terrain culling in Task 6 already accounts for this by treating water columns as `waterLevel`-high neighbors).
- One translucent `InstancedMesh` = one draw call; total scene cost stays at terrain(1) + water(1) + prefab sub-meshes.

- [ ] **Step 1: Create water brick renderer**

Create `src/world/bricks/WaterBrickRenderer.js`.

Target implementation:

```js
import * as THREE from 'three/webgpu'
import { createWaterMaterial } from '../../materials/tsl/waterMaterial.js'

export default class WaterBrickRenderer {
    constructor({ config, brickGeometry }) {
        this.config = config
        this.brickGeometry = brickGeometry
        this.material = createWaterMaterial(config.water)
        this.group = new THREE.Group()
        this.group.name = 'WaterBricks'
        this.mesh = null
        this.capacity = 0
    }

    build(terrainMap) {
        const { width, depth, cellSize, layerHeight, waterLevel } = this.config.terrain

        const cells = []
        for (let z = 0; z < depth; z++) {
            for (let x = 0; x < width; x++) {
                if (terrainMap.getSurfaceCell(x, z).isWater) {
                    cells.push({ x, z })
                }
            }
        }

        if (!this.mesh || cells.length > this.capacity) {
            this.mesh?.dispose()
            this.group.remove(this.mesh)
            this.capacity = Math.ceil(Math.max(cells.length, 1) * 1.2)
            this.mesh = new THREE.InstancedMesh(this.brickGeometry, this.material, this.capacity)
            this.mesh.name = 'WaterBrickInstances'
            this.group.add(this.mesh)
        }

        const matrix = new THREE.Matrix4()
        cells.forEach((cell, i) => {
            matrix.setPosition(
                (cell.x + 0.5) * cellSize,
                waterLevel * layerHeight,
                (cell.z + 0.5) * cellSize
            )
            this.mesh.setMatrixAt(i, matrix)
        })

        this.mesh.count = cells.length
        this.mesh.instanceMatrix.needsUpdate = true

        return this.group
    }

    dispose() {
        this.mesh?.dispose()
        this.material.dispose()
        this.group.parent?.remove(this.group)
        this.group.clear()
        this.mesh = null
        this.capacity = 0
    }
}
```

## Risk Register

| Risk | Impact | V1 Handling |
|------|--------|-------------|
| Brick GLB export breaks conventions (origin, footprint size) | Misaligned grid, floating bricks | Conventions documented in Current Context; `extractBrickGeometry` normalizes scale from bounding box; verify stud alignment in Task 12. |
| Per-instance color unsupported edge cases in WebGPU pipeline | Uniform-colored terrain | `InstancedMesh.setColorAt` is the supported path for node materials; fall back to a per-layer `instancedBufferAttribute` + custom `colorNode` if a renderer regression appears. |
| Missing Blender assets | Scene appears sparse | Keep manifest source names stable and allow missing assets to be skipped without crashing. |
| Visible-layer culling leaves holes at water edges | See-through gaps | Water columns count as `waterLevel`-high neighbors; orbit is clamped so no view angle exposes culled faces; verify in Task 9/12. |
| Biome transition looks muddy | Weak visual identity | Heights blend smoothly, colors hard-cut with seeded dithering; dominant biome drives prefab placement. |
| Terrain generation is too noisy | Poor Lego readability | Real simplex FBM with low octaves, `seaClip` flattening, and floor quantization. |
| Debug regeneration leaks GPU memory | Memory growth | Terrain/water renderers reuse preallocated capacity with dynamic `count`; prefab instancing disposes instance buffers; shared GLB geometry is never disposed by renderers. |
| WebGPU browser support varies | Demo failure | Keep clear error UI in `bootstrap.js` and test in Chrome/Edge with WebGPU enabled. |
