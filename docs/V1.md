# Lego Stylized Nature V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a WebGPU + TSL foundation for a Lego-style stylized nature scene with noise-based terraced terrain, hand-guided biome regions, Blender-exported terrain tiles and nature prefabs, and four polished biomes.

**Architecture:** V1 uses a fixed-size mainland instead of infinite terrain streaming. Terrain height is generated from noise, biome ownership is generated from hand-guided cell regions with blended borders, and Blender-exported prefabs are placed by data-driven rules. `World` becomes an orchestrator; terrain, biome, tile assets, prefab placement, materials, and debug panels are separated into focused modules.

**Tech Stack:** Vite, Three.js WebGPU (`three/webgpu`), TSL (`three/tsl`), Tweakpane, GLTF/GLB assets exported from Blender.

---

## V1 Scope

V1 builds a controllable showcase island/mainland, not a full open-world engine.

Included:

- Fixed terrain size, recommended default `96 x 96` terrain cells.
- Noise-based terraced height generation inspired by the Blender Geometry Nodes prototype.
- Four biomes: `grassland`, `forest`, `desert`, `tundra`.
- Cell-noise biome mask with soft transition weights between nearby biomes.
- Per-biome palette, terrain layer rules, asset pools, and placement density.
- Blender-exported terrain tiles and nature prefabs loaded as GLB resources.
- Instanced or cloned placement path for terrain tiles and prefabs, with a clear upgrade path to heavier batching.
- TSL materials for Lego surface, water, snow/ice accents, and foliage.
- Debug panels for seed, terrain height, biome mask, placement density, and material colors.

V1 excludes:

- Infinite chunk streaming.
- Player controller, combat, inventory, mining, or survival mechanics.
- Climate simulation based on temperature and humidity.
- Procedural mesh generation for cactus, bamboo, trees, flowers, or Lego bricks.
- Full biome ecology rules.
- Large-scale asset optimization beyond basic reuse, disposal, and grouping.

## Data Flow

```txt
WorldConfig
  -> Resources loads terrain tiles and biome prefabs
  -> BiomeRegistry loads four biome definitions
  -> TerrainGenerator creates HeightField from seeded noise
  -> BiomeMaskGenerator creates biome id and biome weights per cell
  -> SurfaceClassifier labels cells as water, shore, slope, surface, subsurface
  -> TilePaletteResolver selects tile asset ids from biome and layer data
  -> LayeredTerrainBuilder creates tile placement records
  -> TileInstanceBuilder builds visible terrain groups
  -> PrefabPlacer scatters Blender prefab records on valid top surfaces
  -> WaterSurface renders a stylized water plane or tile layer
  -> Debug panels expose seed and rule controls
```

## Naming Conventions

- Biome ids use lower camel case: `grassland`, `forest`, `desert`, `tundra`.
- Terrain tile ids describe visual role: `grassTop1x1`, `dirtSide1x1`, `sandTop1x1`, `snowTop1x1`, `waterTile1x1`.
- Prefab ids describe biome and object family: `desertCactusSmall`, `forestPineSmall`, `grasslandFlowerWhite`, `tundraSnowRock`.
- Placement records use world cell coordinates: `{ x, y, z, biomeId, tileId }`.
- Runtime classes use PascalCase filenames and named methods with explicit responsibilities.

```js
export function hash2(x, z, seed = 0) {
    let h = seed ^ Math.imul(x + 374761393, 668265263) ^ Math.imul(z + 69069, 2246822519)
    h = (h ^ (h >>> 13)) >>> 0
    h = Math.imul(h, 1274126177) >>> 0
    return (h ^ (h >>> 16)) >>> 0
}

export function random01(x, z, seed = 0) {
    return hash2(x, z, seed) / 4294967295
}

export function pickWeighted(entries, randomValue) {
    let total = 0
    for (const entry of entries) {
        total += entry.weight
    }

    if (total <= 0) {
        return entries[0]?.value
    }

    let cursor = randomValue * total
    for (const entry of entries) {
        cursor -= entry.weight
        if (cursor <= 0) {
            return entry.value
        }
    }

    return entries[entries.length - 1]?.value
}

export function snapValue(value, step) {
    return Math.round(value / step) * step
}
```

Target implementation:

```js
export default class BiomeMaskGenerator {
    constructor(config) {
        this.config = config
    }

    generate() {
        const { width, depth } = this.config.terrain
        const cells = []

        for (let z = 0; z < depth; z++) {
            const row = []
            for (let x = 0; x < width; x++) {
                row.push(this.getCellBiome(x, z))
            }
            cells.push(row)
        }

        return cells
    }

    getCellBiome(x, z) {
        const regions = this.config.biomes.regions
        const scores = regions.map((region) => {
            const dx = x - region.center[0]
            const dz = z - region.center[1]
            const distance = Math.sqrt(dx * dx + dz * dz)
            const normalized = distance / region.radius
            const score = Math.max(0, 1 - normalized) * region.weight
            return { id: region.id, score }
        }).filter((entry) => entry.score > 0)

        if (scores.length === 0) {
            return { biomeId: 'grassland', weights: { grassland: 1 } }
        }

        scores.sort((a, b) => b.score - a.score)
        const selected = scores.slice(0, 2)
        const total = selected.reduce((sum, entry) => sum + entry.score, 0)
        const weights = {}

        for (const entry of selected) {
            weights[entry.id] = entry.score / total
        }

        return { biomeId: selected[0].id, weights }
    }
}
```

```js
export default class HeightField {
    constructor(width, depth) {
        this.width = width
        this.depth = depth
        this.values = Array.from({ length: depth }, () => Array.from({ length: width }, () => 0))
    }

    get(x, z) {
        return this.values[z]?.[x] ?? 0
    }

    set(x, z, value) {
        if (this.values[z]) {
            this.values[z][x] = value
        }
    }

    getSlope(x, z) {
        const center = this.get(x, z)
        const neighbors = [
            this.get(x + 1, z),
            this.get(x - 1, z),
            this.get(x, z + 1),
            this.get(x, z - 1)
        ]
        return Math.max(...neighbors.map((height) => Math.abs(center - height)))
    }
}
```

```js
import HeightField from './HeightField.js'
import TerrainMap from './TerrainMap.js'
import SurfaceClassifier from './SurfaceClassifier.js'
import { random01 } from '../../utils/random.js'

export default class TerrainGenerator {
    constructor({ config, biomeMaskGenerator, biomeBlender }) {
        this.config = config
        this.biomeMaskGenerator = biomeMaskGenerator
        this.biomeBlender = biomeBlender
        this.surfaceClassifier = new SurfaceClassifier(config)
    }

    generate() {
        const biomeCells = this.biomeMaskGenerator.generate()
        const heightField = this.generateHeightField(biomeCells)
        const surfaceCells = this.surfaceClassifier.classify(heightField)
        return new TerrainMap({ heightField, biomeCells, surfaceCells })
    }

    generateHeightField(biomeCells) {
        const terrain = this.config.terrain
        const field = new HeightField(terrain.width, terrain.depth)

        for (let z = 0; z < terrain.depth; z++) {
            for (let x = 0; x < terrain.width; x++) {
                const biomeCell = biomeCells[z][x]
                const heightOffset = this.biomeBlender.blendTerrainParam(biomeCell.weights, 'heightOffset', 0)
                const heightMagnitude = this.biomeBlender.blendTerrainParam(biomeCell.weights, 'heightMagnitude', 1)
                const n = this.fbm(x, z)
                const normalized = Math.max(0, Math.min(1, 0.5 + n * 0.5))
                const height = Math.floor((normalized * terrain.maxHeight * heightMagnitude) + terrain.waterLevel + heightOffset)
                field.set(x, z, Math.max(1, Math.min(terrain.maxHeight, height)))
            }
        }

        return field
    }

    fbm(x, z) {
        const terrain = this.config.terrain
        let value = 0
        let amplitude = 1
        let frequency = 1 / terrain.noiseScale
        let totalAmplitude = 0

        for (let octave = 0; octave < terrain.noiseOctaves; octave++) {
            const sample = random01(
                Math.floor(x * frequency * 1000),
                Math.floor(z * frequency * 1000),
                this.config.seed + octave * 101
            ) * 2 - 1
            value += sample * amplitude
            totalAmplitude += amplitude
            amplitude *= terrain.noiseGain
            frequency *= terrain.noiseLacunarity
        }

        return value / totalAmplitude
    }
}
```

```js
export function canPlacePrefab(rule, surfaceCell) {
    if (surfaceCell.isWater || surfaceCell.isShore) {
        return false
    }
    if (rule.minHeight !== undefined && surfaceCell.height < rule.minHeight) {
        return false

    }

export function makePrefabTransform({ x, z, height, config, seed }) {
    const { cellSize, layerHeight } = config.terrain
    const rotationStep = config.placement.rotationStep
    const scaleStep = config.placement.scaleStep
    const rotation = snapValue(random01(x, z, seed + 17) * Math.PI * 2, rotationStep)
    const scale = snapValue(0.9 + random01(x, z, seed + 23) * 0.2, scaleStep)

    return {
        position: [x * cellSize, height * layerHeight, z * cellSize],
        rotationY: rotation,
        scale
    }
}
```

```js
import * as THREE from 'three/webgpu'
import { random01 } from '../../utils/random.js'

export default class PrefabPlacer {
    constructor({ config, biomeRegistry, prefabRegistry }) {
        this.config = config
        this.biomeRegistry = biomeRegistry
        this.prefabRegistry = prefabRegistry
        this.group = new THREE.Group()
        this.group.name = 'BiomePrefabs'
    }

    build(terrainMap) {
        this.dispose()
        this.group = new THREE.Group()
        this.group.name = 'BiomePrefabs'

        const { width, depth } = this.config.terrain
        for (let z = 0; z < depth; z++) {
            for (let x = 0; x < width; x++) {
                const biomeCell = terrainMap.getBiomeCell(x, z)
                const biome = this.biomeRegistry.get(biomeCell.biomeId)
                const surfaceCell = terrainMap.getSurfaceCell(x, z)

                for (const rule of biome.prefabs) {
                    if (!canPlacePrefab(rule, surfaceCell)) {
                        continue
                    }
                    if (random01(x, z, this.config.seed + rule.id.length) > rule.density) {
                        continue
                    }

                    const prefab = this.prefabRegistry.get(rule.id)
                    const sourceScene = prefab?.asset?.scene
                    if (!sourceScene) {
                        continue
                    }

                    const transform = makePrefabTransform({
                        x,
                        z,
                        height: surfaceCell.height + 1,
                        config: this.config,
                        seed: this.config.seed
                    })

                    const instance = sourceScene.clone(true)
                    instance.position.fromArray(transform.position)
                    instance.rotation.y = transform.rotationY
                    instance.scale.setScalar(transform.scale)
                    this.group.add(instance)
                }
            }
        }

        return this.group
    }

    dispose() {
        this.group?.parent?.remove(this.group)
        this.group?.clear()
    }
}
```
